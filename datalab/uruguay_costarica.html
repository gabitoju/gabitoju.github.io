Tweets Uruguay - Costa Rica

Twittear

¿Cuántos tweets se generaron durante el partido con el hashtag #URU? ¿Cuántos de ellos fueron durante el primero tiempo y cuántos durante el segundo? ¿Qué usuarios fueron los que generaron más contenido? ¿Qué pasó cuándo explusaron al "Mono" Pereira?

Usando la Streaming API de Twitter, ejecuté un script que buscaba y almacenaba todos los tweets con mención al hashtag #URU para luego responder las preguntas que me planeto más arriba. El script se puede descargar de acá. Este script depende de Tweepy y almacena los tweets en MongoDB usando PyMongo y está basdo en éste. El rango de tiempo fue desde las 16:00 (inicio del partido) hasta las 18:06 (algunos minutos luego del fin para tener un universo más interesante de tweets).
